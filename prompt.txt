OPSEC Strategies by Stealth Level
Ghost Mode (Maximum Stealth):

Sleep: 30-60 seconds between operations
Jitter: 0.5-0.8 randomness
Network: 1-3 packets only
LOLBins: Mandatory
Encryption: Always
Delays: Random 1-5 minutes

Silent Mode (High Stealth):

Sleep: 10-20 seconds
Jitter: 0.3-0.5
Network: <10 packets
LOLBins: Preferred
User simulation: Yes

Quiet Mode (Balanced):

Sleep: 3-5 seconds
Jitter: 0.2-0.3
Network: <50 packets
LOLBins: When possible

Normal Mode (Standard):

Sleep: 0-1 second
No restrictions
