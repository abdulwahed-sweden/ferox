
## Part A: Credential Vault

### Layout
Two-column layout:
- Left: Credential list (scrollable)
- Right: Selected credential details

### Credential List (Left)
Grouped by type with expandable sections:
- 🔑 Plain Text Passwords (count)
- #️⃣ NTLM Hashes (count)
- 🎫 Kerberos Tickets (count)
- 🔐 SSH Keys (count)
- 🌐 Cloud Credentials (count)
- 🎟️ Tokens & Certificates (count)

Each credential card shows:
- Username (bold)
- Domain (if applicable)
- Source hostname
- Collection timestamp
- Sensitivity badge (Critical/High/Medium/Low)
- Reusability indicator (can be used for lateral movement)

### Search & Filters (Top)
- Search by username, hostname, or domain
- Filter by type (checkboxes)
- Filter by sensitivity
- Filter by source session
- "Show reusable only" toggle
- Sort by: newest, oldest, sensitivity

### Credential Actions
For each credential:
- 👁️ Show/Hide secret (redacted by default)
- 📋 Copy to clipboard
- 🎯 Test against targets
- ➡️ Use for lateral movement
- 🗑️ Delete
- 📝 Add note

### Bulk Actions (Top Bar)
When credentials selected:
- Test all against discovered targets
- Export selected (JSON/CSV/Hashcat format)
- Delete selected
- Mark as high-priority

### Credential Details Panel (Right)
When clicking credential, show:
- Full credential information
- Redacted secret with "Show" button
- Source session details
- When collected
- Reuse history (which sessions used this)
- Password strength meter (if applicable)
- Related credentials (same user, different systems)
- MITRE technique: T1555, T1552, T1003

### Intelligence Panel
Show analysis:
- Password reuse detected (X passwords used on Y systems)
- Domain admin credentials found (highlight in red)
- Cloud admin access (highlight in orange)
- Weak passwords detected
- Recommendations for privilege escalation

## Part B: MITRE ATT&CK Dashboard

### Interactive MITRE Matrix
Display tactics (columns) and techniques (rows):

**Tactics (top row):**
Initial Access | Execution | Persistence | Privilege Escalation | 
Defense Evasion | Credential Access | Discovery | 
Lateral Movement | Collection | Exfiltration | C&C

**Heat Map Colors:**
- 🟢 Green: Used successfully, low detection
- 🟡 Yellow: Used, medium detection risk
- 🟠 Orange: Used, high detection risk
- ⚪ Gray: Available but not used
- ⬛ Black: Not available

**Click on technique:**
Show modal with:
- Technique ID and name
- Full description
- When used (timestamps)
- Which sessions used it
- Success rate
- Detection rate
- Related sub-techniques
- OPSEC considerations

### Coverage Statistics (Top)
Cards showing:
- Overall coverage: 78% (with progress bar)
- Techniques used: 24/156
- Tactics covered: 11/14
- Detection risk: Medium (average across all)

### Tactic Detail Cards (Below Matrix)
For each tactic, show card:
- Tactic name and coverage percentage
- Techniques used from this tactic
- Mini progress bar
- Click to filter matrix

### Timeline View (Toggle)
Alternative view showing techniques used over time:
- Horizontal timeline
- Each technique as a dot on timeline
- Color by tactic
- Size by importance
- Hover for details

### OPSEC Overlay
Toggle to show detection risk:
- Green overlay: Low risk techniques
- Red overlay: High risk techniques
- Recommendations for safer alternatives

- Lazy load components with React.lazy

**WebSocket Optimization:**
- Batch updates (collect events for 100ms before updating UI)
- Only update visible components
- Unsubscribe from sessions when not viewing

**Bundle Size:**
- Code splitting by route
- Tree shaking unused code
- Compress images
- Lazy load cytoscape only on network page
